## Download File Kali Linux
  1. Akses situs www.kali.org
  2. Klik tombol download
  3. Klik menu Installer Image'
  4. Download file Kali Linux Installer x64
     <img width="952" alt="21" src="https://github.com/user-attachments/assets/31bf70d8-34dd-4faa-a13f-e0a401195bc3">
     
## Installasi Kali Linux pada Virtual Box
  1. Download Virtual Box dari situs www.virtualbox.org , kemudian install
  2. Buka aplikasi Virtual Box
  3. Klik `new` pada bagian atas

     <img width="960" alt="1" src="https://github.com/user-attachments/assets/741e5107-1db1-4397-af82-0e2e3862a2b8">
  4. Isi bagian **Nama** sesuai sistem operasi yang akan diinstall. Pada bagian **Iso Image** masukan file iso Kali Linux yang sudah didownload tadi, Kemudian klik `Next`

     <img width="960" alt="2" src="https://github.com/user-attachments/assets/1f937eef-6437-450c-868e-8ea18eec004b">

  5. Sesuaikan Hardwawre dengan spesifikasi laptop/pc kalian. Kemudian klik `Next`

     <img width="960" alt="3" src="https://github.com/user-attachments/assets/84e8e1e5-d0bb-4ff5-91fd-0a350e57d75e">
    
  6. Klik `Next`

     <img width="960" alt="4" src="https://github.com/user-attachments/assets/453eac9e-f06d-41bd-8a6b-2221835f301e">

  7. Klik `Finish`

     <img width="960" alt="5" src="https://github.com/user-attachments/assets/7bb123ff-88e6-4ee1-b165-17565305138d">

  8. Klik `Settings` untuk menkonfigurasi internet didalam sistem operasinya

     <img width="960" alt="6" src="https://github.com/user-attachments/assets/3d7d84da-63e7-45fa-99e3-a5c5c6e94462">

  9. Klik menu `Network`, Kemudian ubah pada bagian **Attached to** menjadi **Bridge Adapter** dan pada bagian **Advanced** pilih **Allow All** dibagian **Promiscuous Mode**, Kemudian klik **Ok**

     <img width="960" alt="7" src="https://github.com/user-attachments/assets/3301b71b-dc86-44f9-97f7-ee3b25249f05">

  10. Kemudian klik **Start** untuk memulai sistem operasi

      <img width="960" alt="8" src="https://github.com/user-attachments/assets/810d7372-fbde-489d-8c1a-fa05651c5517">

  11. Jika sistem operasi sudah berjalan, Klik **Graphical Install**

      <img width="960" alt="9" src="https://github.com/user-attachments/assets/a542f8df-341f-44f2-a1db-a4687b694767">

  12. Pilih Bahasa anda, Kemudian klik `Continue`

      <img width="960" alt="10" src="https://github.com/user-attachments/assets/49e11a38-2217-4f75-be62-a93ecd6d187f">

  14. Pilih Lokasi anda, Kemudian klik `Continue`

      <img width="960" alt="11" src="https://github.com/user-attachments/assets/45704014-10cf-455c-ae36-eed0e69e0fef">

  15. Pilih Benua anda, Kemudian klik `Continue`

      <img width="960" alt="12" src="https://github.com/user-attachments/assets/8b77e89c-0aab-4c30-b66c-2286762f4cda">

  16. Pilih Negara anda, Kemudian klik `Continue`

      <img width="960" alt="13" src="https://github.com/user-attachments/assets/b1399be5-0b10-4023-b013-3dbd40cd5f78">

  17. Pilih Layout Keyboard anda, Kemudian klik `Continue`

      <img width="960" alt="14" src="https://github.com/user-attachments/assets/391537f3-20c6-4b22-9d99-80fa97e64548">

  18. Tunggu installasi sampai selesai, diproses ini memakan waktu yang lama sekitar 20-30 menit

      <img width="960" alt="15" src="https://github.com/user-attachments/assets/11661ee3-2bbc-4f0b-b0ca-4747301fb0e2">

  19. Pada bagian ini, pilih **Yes** kemudian klik `Continue`

      <img width="960" alt="16" src="https://github.com/user-attachments/assets/aa8f29dd-1121-434b-abfb-c27e65c648c7">

  20. Setelah ini, proses akan berjalan lagi

      <img width="960" alt="17" src="https://github.com/user-attachments/assets/1e35e45b-5b72-420e-a0f0-c9d5abcacdd7">

  21. Jika semua selesai, maka sistem meminta untuk **Reboot Sistem**, Kemudian klik `Continue`, Tunggu hingga **Booting** sistem selesai

      <img width="960" alt="18" src="https://github.com/user-attachments/assets/177cb82b-7227-434f-b993-1c425ff89934">

  22. Sistem meminta memasukan **Username** dan **Password** yang sudah dikonfigurasi pada saat installasi tadi

      <img width="960" alt="19" src="https://github.com/user-attachments/assets/787316ff-b6d3-4707-8d7f-acda85318de7">

  23. Selamat!!!, anda berhasil menginstall **Kali Linux** pada **Virtual Box**

      <img width="956" alt="20" src="https://github.com/user-attachments/assets/62ad77bb-7a09-4df9-9976-0d927f276cdb">
